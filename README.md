**Weather App**
This is a simple weather app that provides current weather information and a 24-hour weather forecast for any city. The app fetches data from the OpenWeatherMap API to display the temperature, weather description, and an hourly forecast with corresponding weather icons.

**Features**
1. Current Weather: Displays the current temperature, weather description, and weather icon for a searched city.
2. Hourly Forecast: Shows the hourly weather forecast for the next 24 hours, with temperature and weather icons.
3. Responsive Design: The app is responsive, making it accessible on devices of various screen sizes, from mobile phones to tablets and desktops.

**Technologies Used**
1. HTML5
2. CSS3
3. JavaScript
4. OpenWeatherMap API

**How to Use**
1. Open the app in a web browser.
2. Enter the name of the city in the search input field.
3. Click the "Search" button.
4. The app will display the current temperature, weather description, and weather icon for the city, along with the hourly forecast for the next 24 hours.

**App Demo**
Once you've set up the app and entered a city, you will see:
  Current Temperature (in Celsius)
  Weather Description (e.g., sunny, cloudy)
  Weather Icon (based on the weather)
  Hourly Forecast for the next 24 hours, showing the temperature and an icon representing the weather condition.
