# Weather App

A simple web-based Weather App built using **HTML**, **CSS**, and **JavaScript** that allows users to get the current weather information for any city. The app fetches weather data from a public API and displays the temperature, humidity, wind speed, and more.

## Features

- Search for current weather in any city
- Displays temperature, humidity, wind speed, and weather conditions (e.g., sunny, rainy, etc.)
- User-friendly interface with clear and responsive design
- Dynamic background change based on weather conditions
- Simple and intuitive UI with search functionality

## Technologies Used

- **HTML**: Used to create the structure of the weather app, including the input field, buttons, and display area.
- **CSS**: Used for styling the app, making it responsive and visually appealing.
- **JavaScript**: Used to handle fetching data from the weather API and dynamically updating the UI based on user input.
- **Weather API**: The app uses the OpenWeatherMap API (or another weather API of your choice) to retrieve real-time weather data.
